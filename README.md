# Himasha Jayasinhe - Personal Portfolio Website

This is a professional, responsive, and "supiri" (superb) personal portfolio website built with HTML, CSS, JavaScript, and Tailwind CSS.

## Features

- **Modern Design:** Dark mode aesthetic with glassmorphism effects and smooth animations.
- **Responsive:** Works perfectly on mobile, tablet, and desktop.
- **Fast:** No heavy frameworks, just pure HTML/JS with Tailwind CDN.
- **Interactive:** Smooth scrolling, reveal animations on scroll, and interactive hover effects.

## How to Edit

1. **Text Content:** Open any `.html` file (e.g., `index.html`, `about.html`) in a text editor (like VS Code or Notepad) and change the text.
2. **Images:** currently using Unsplash placeholder images. To use your own:
   - Save your images in `assets/images/`.
   - Update the `src` attributes in the HTML files to point to your images (e.g., `src="assets/images/my-photo.jpg"`).
3. **Colors/Styles:** You can modify `assets/css/styles.css` or change Tailwind classes in the HTML classes.

## Project Structure

- `index.html` - Home page
- `about.html` - About Me page
- `services.html` - Services page
- `portfolio.html` - Portfolio Gallery
- `contact.html` - Contact page
- `assets/`
  - `css/styles.css` - Custom styles
  - `js/script.js` - JavaScript for interactivity
